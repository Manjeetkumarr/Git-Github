Deep Learning Project

This project demonstrates the application of Convolutional Neural Networks (CNNs) for image classification tasks using deep learning. The primary objective is to classify images of cats and dogs using the popular Kaggle Cats vs Dogs dataset. I use TensorFlow and Keras to build and train a deep learning model that learns to differentiate between cat and dog images. The model is trained on labeled image data, evaluated using validation metrics, and used to predict the class of unseen images.
